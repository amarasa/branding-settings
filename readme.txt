A plugin that allowed you to create brand settings in custom fields and then display through a shortcode.

== Description ==

This plugin allows you to configure branding details in the settings and then use a shortcode anywhere on the site to display the values. This makes it easy to make changes sitewide, without having to change branding in multiple areas.
== Frequently Asked Questions ==

= Does this require any 3rd party plugins? =

Yes, you will need to have Advanced Custom Fields installed in order for this plugin to work.
